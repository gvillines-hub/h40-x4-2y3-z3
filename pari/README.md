# PARI/GP scripts (PARI 2.15.4)

Raw working scripts from the campaign, in chronological order. Each is
self-contained (`gp -q scriptname.gp`). Highlights:

* 01-04: the parametrization over Z[cbrt2], cover construction, scaling/charts,
  and local solubility screen producing the six curves.
* 05: the norm-form identity N(gamma(x)) = C3(x).
* 06: factorization of C3 over K and Frobenius/Euler-factor data.
* 07: charpoly factorization over Q(sqrt(-2)) at split primes (pi^2 structure).
* 08: refutation battery for real multiplication by small quadratic fields.
* 09-13: numerical involutions and twist maps; field identification via lindep.
* 14: Hecke character data at 26 primes (trace vanishing at inert primes;
  16/16 split primes <= 200 verified with constant terms +-pi^2, +-pibar^2).
* 15-17: tangent-representation entry fields (deg 8 and deg 6) and the
  degree-24 compositum analysis.
* 18: Chabauty prime selection (split primes with rational Weierstrass points;
  selects p = 43, backups 89, 113).
